A minimal real-time audio gain plugin built using the JUCE framework.  
Developed as part of my exploration into audio DSP and plugin development for music technology.

## Features
- Real-time gain control with smooth parameter changes
- Low-latency audio processing
- Clean and minimal UI
- Supports AU / VST formats
- Tested in Logic Pro

## Tech Stack
- C++
- JUCE Framework
- Audio DSP
- Xcode (macOS)
- Logic Pro (AU testing)

## How to Build
1. Download and extract `SimpleGain.zip`
2. Open `SimpleGain.jucer` in Projucer
3. Export to Xcode
4. Build AU/VST target
5. Scan plugin in your DAW

## Motivation
This project was built to understand:
- JUCE AudioProcessor lifecycle
- Real-time audio processing
- Parameter smoothing and automation
- Plugin UI fundamentals
